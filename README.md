# 安装( Linux, Mac)

最快的安装方式:

## 1. 前提

你安装的是 vim 7.3+ 版本。

## 2. 复制文件

就是把两个配置文件，都放到home目录下。

```
$ unzip vim_folder.zip      # 解压缩 这个文件.
$ cp .vim ~ -r              # 把 解压缩出来的.vim 文件夹放到 HOME目录
$ cp .vimrc ~               # 把 配置文件 .vimrc 放到 HOME 目录.
```

就安装好了.
